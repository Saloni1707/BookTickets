# 🎟️ Ticket Booking System (Backend - Java)
---
This is a backend focused project built to understand how java is used for backend
## 🚀 Features

- ✅ View available events/shows
- ✅ Book tickets 
- ✅ Cancel tickets
- ✅ View booking history
- ✅ Admin controls for event creation

---
